# ImgUp  
Image Memo Window Application  
> Target Framework : .NET Framework 4.6.1  
  
## Functions
+ Load image from Clipboard and show up. (Hotkey : Alt + Shitf + 0 ~ 9)  
+ Save images and location of memo.  
+ Memo Options
  * TopMost
  * Minimize
  * Save image
  * Copy
  * Hide 
  * Erase
